# Time-Dependent Ginzburg-Landau Simulation
---

This is a passion project done in C++ attempting to use OpenGL to render a high-performance simulation of a basic superconductor by solving the Time-Dependent Ginzburg-Landau equation using a 5-point discretization scheme, and a Courant-Friedrichs-Lewy compliant timestep.  The goal is to demonstrate an understanding of the computational methods as well as the field theories underlying this process.

Within this repo you will find two main sections:

1. The Simulation -- A TDGL simulation coded in C++, displaying two diagrams.  The left diagram is a phase map of the system
